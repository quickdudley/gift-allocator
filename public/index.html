Introduction

In order to simplify our Christmas festivities last year: my sister suggested that we could arrange things so each family member only gave and received one present each. At the time I wrote a command line program in Haskell to allocate each family member another one to give a present to. This page is essentially the same thing: but in the form of a web app in React.js

Usage

Type the name of each participant into the text box labelled "Name". If any participant requires assistance (i.e. due to visual impairment): check the box labelled "Assistance required" and check the names of each person able to assist. When you've finished editing a participant's name and details: you can click add. Once everyone is entered: click "Do allocations", download the generated text files, and send them to the recipients named.
